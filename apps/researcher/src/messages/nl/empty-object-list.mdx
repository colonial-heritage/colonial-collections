**Deze lijst bevat nog geen objecten.** Maar je kunt ze eenvoudig toevoegen:

1. Maak een account aan of log in. Hier vind je [informatie over het hebben van een account](/sign-up).
2. Zoek een object met behulp van [de zoekpagina](/objects) en klik op een object.
3. Klik op de detailpagina op de knop "Object aan lijst toevoegen".
4. Selecteer de *{props.name}* lijst.
